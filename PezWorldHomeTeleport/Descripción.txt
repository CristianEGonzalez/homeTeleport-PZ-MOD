[h2][MOD INFORMATION][/h2]

Teleporter craftable item for Multiplayer servers / Only players.

Right clicking on the item opens a contextual menu with 2 options:
Set Home: To save your current location.
Teleport to Home: To return to that location.

It is possible to configure the Cooldown time (in hours) for reuse from the sandbox options.

    recipe Make Teleporter
    {
        Aluminum = 2,
        Amplifier = 1,
	ElectricWire = 2,
	ElectronicsScrap = 4,
	RadioReceiver = 1,
	RadioTransmitter = 1,
	Battery = 1,
        Result: Teleporter,
        Time: 30.0,
    }

[h2][INFORMACIÓN DEL MOD][/h2]

Artículo fabricable "Teletransportador" para servidores Multiplayer / Un Jugador.
Al hacer click derecho en el item se abre un menú contextual con 2 opciones:
Set Home: Para guardar tu ubicación actual.
Teleport to Home: Para regresar a esa ubicación.

Es posible configurar el tiempo de Cooldown para su reutilización desde las sandbox options.

    receta Crear Teletransportador
    {
        Papel de aluminio = 2,
        Amplificador = 1,
	Cable eléctrico = 2,
	Chatarra electrónica = 4,
	Radiorreceptor = 1,
	Radiotransmisor = 1,
	Pila = 1,
        Resultado: Teletransportador,
        Tiempo: 30.0,
    }
